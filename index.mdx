## Autenticação

Todas as consultas \*\*exigem \*\*autenticação por API Token. Para obter o seu, entre em contato por email.

Todas as consultas devem incluir um **\_header \_**de "**Authorization**" com o valor "**Bearer SEU_TOKEN_AQUI**".

## Sintaxe de Consulta

Nossas séries temporais são consolidadas de forma a possuir endpoints específicos, cada um com seus parâmetros. Entretanto, possuímos uma rota focada em disponibilizar todas as séries de forma centralizada, que deve ser consultada usando uma sintaxe padronizada.

### Estrutura da API centralizada

<Info>
  [**http://orion.thiago-simoes.com/api/v1/series/:identificador**](http://orion.thiago-simoes.com/api/v1/series/:identificador)
</Info>

### Identificação da Série

As séries são identificadas por um identificador compos por três partes:

1. **Fonte primária dos dados**
2. **Base de dados**
3. **Série**

**_O identificador é a concatenação desses campos.Importante: apenas a identificação única da série é case-sensitive._**

**Exemplo**

<Check>
  \*\*BCB_SGS_12 \*\*

  \
  **Fonte:** Banco Central do Brasil

  **Base de Dados:** SGS

  **Série:** 12 - "Taxa de juros - CDI"

  \
  \
  **ANBIMA_CARTEIRAS-TEORICAS_IMA-B**

  **Fonte**: Anbima

  **Base de Dados**: Carteiras Teóricas

  **Série**: IMA-B
</Check>

<Danger>
  **SGS_12**

  **Fonte**: ?

  **Base de Dados**: SGS

  **Série**: 12 - "Taxa de juros - CDI"

  \
  **ANBIMA_IMA-B**

  **Fonte**: ?

  **Base de Dados**: Anbima

  **Série**: 12 - "Taxa de juros - CDI"
</Danger>

## Parâmetros

- **dataInicial**\
  **nome**    : data inicial\
  **formato**    : _dd-mm-yyyy_\
  **explicação**: a data inicial dos retornos\
  **obrigatório**: não
- **dataFinal**\
  **nome**    : data final\
  **formato**    : _dd-mm-yyyy_\
  **explicação**: a data final dos retornos\
  **obrigatório**: não
- **colunas**\
  **nome**    : colunas selecionadas\
  **formato**    : _col1,col2,col3_\
  **explicação**: lista de colunas a serem retornadas\
  **obrigatório**: não
- **colunasExcluir**\
  **nome**    : colunas a serem excluídas\
  **formato**    : _col1,col2,col3_\
  **explicação**: lista de colunas a serem excluídas\
  **obrigatório**: não

**Importante:** consultas sem os parâmetros retornarão todas as colunas, no intervalo padrão de tempo (2 anos).
